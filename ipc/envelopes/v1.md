# IPC Envelope v1

## Paths
- inbox: `/data/lucy_ipc/inbox/`
- outbox: `/data/lucy_ipc/outbox/`
- deadletter: `/data/lucy_ipc/deadletter/`

## JSON envelope fields
- `version` (string): `"v1"`
- `correlation_id` (string)
- `received_ts` (RFC3339 string)
- `payload` (object): original request body
- `headers_subset` (object): selected HTTP headers
- `source_ip` (string)
- `status` (string): `"accepted" | "duplicate" | "deadletter"`
- `reason` (string, optional)

## Filename
- `{correlation_id}.json`
